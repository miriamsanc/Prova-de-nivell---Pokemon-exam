# Prova de nivell
Tenemos un registro de un equipo pokemon.

De cada pokemon conocemos:
- Nombre
- Tipo (pueden ser 2)
- Vida máxima
- Ataques (solo conocemos el nombre)

Registramos 4 pokemons por lo menos. 

Después:
- Dado un tipo, devolver pokemons de ese tipo.
- Devuelve pokemon con menos vida.
- Dado un ataque, haz que el pokemon "realice" ese ataque.
